## Projects :sparkles:

### 1. Geospatial AI & Modeling 🌊

#### 🌊 도동실(DoDongSil): 여수 다도해 해양쓰레기 예측 시스템 (2026)
- **개요**: 여수시 다도해(365개 섬)의 해양쓰레기를 **사후 수거 → 선제 예측**으로 전환하는 AI 시스템. 국내 연안 분야 최초로 **MOHID 입자추적 + GNN + LSTM** Grey-box 하이브리드 구조 시도. **최우수상 수상**
- **기술 환경**: Python, PyTorch, PyTorch Geometric(GAT), QGIS, GeoPandas, NetworkX, Streamlit
- **주요 활동**:
  - QGIS 기반 해안선 세그먼트 분할 및 127개 방출점 공간 분석, 섬 반경 1km 버퍼 × 입자 궤적 중첩으로 정량적 유입량 산정
  - 해류 방향 가중 엣지로 섬·방출점을 연결한 **Graph Attention Network(GAT)** + LSTM 인코더 구현
  - MOHID 시뮬레이션 상한선 기반 **물리 제약 손실 함수**로 물리적으로 타당한 예측 유도
  - 데이터 희소 환경(레이블 23개)에서 **LOOCV** 검증, 남해안 32개 관측소 사전학습 → 여수 전이학습
  - Streamlit 5탭 대시보드(Overview·Predictions·Route Optimization·Analysis·Report) 구축
- **Link**: [GitHub Repository](https://github.com/JeongWon4034/yeosu)

#### 🌊 IPAP: 지능형 오염원 통합 분석 플랫폼 (2025)
- **개요**: 지하수 오염 발생 시 `어디서(Where) → 어떻게(Origin) → 누가(Who)`로 이어지는 3단계 원인 규명 프레임워크. **2025 대국민 지하수 빅데이터 공모전 환경부장관상(최우수상) 수상작**
- **기술 환경**: Python, PyTorch(GNN), Flopy/MODFLOW, Scikit-learn, React
- **주요 활동**:
  - 분산된 9종의 오염원 데이터를 '시군구' 단위로 연계·표준화하고 복합 오염 지표(불투수층 면적·총 오염 부하량 등) 피처 엔지니어링
  - **Flopy(MODFLOW)** 기반 역방향 유동 모델링으로 오염 후보 권역 도출
  - **GNN** 기반 공간 관계·패턴 유사도 분석으로 오염원 유형 추론
  - 물리 기반 수리 모델링 + 데이터 기반 딥러닝의 상호 보완적 결합(Hybrid)으로 XAI 지향 분석
- **Link**: [GitHub Repository](https://github.com/JeongWon4034/underwater)
